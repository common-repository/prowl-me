A plugin making your visitors able to write you a direct message using Prowl, a iPhone push application.

== Description ==

Using “Prowl Me” users can send you a message, through the iPhone push application “Prowl”.

Usage examples:
<ul>
    <li>[prowl-me]</li>
</ul>

== Installation ==

Easy installation:<br />
1. Upload the files into the `/wp-content/plugins/` directory<br />
2. Activate the plugin through the 'Plugins' menu in WordPress<br />
3. Place [prowl-me] in your MCE editor (see Description).<br />
